## 一、创始人与核心管理层

### 1.1 李西廷（Li Xiting）—— 董事长 / 联合创始人

| 项目 | 详情 |
|------|------|
| **出生** | 1951年，安徽省宿州市砀山县 |
| **国籍** | 新加坡（2018年迈瑞IPO前入籍） |
| **学历** | 中国科学技术大学物理系低温物理专业学士（1973-1976） |
| **持股** | 26.98%（通过 Smartco Development Limited，BVI注册） |
| **2025年年薪** | 1,880.71万元（同比降24.6%，降薪超600万） |
| **Forbes身家** | ~130亿美元（2025/26），新加坡第5，较2021年巅峰（~230亿美元，新加坡首富）缩水约700亿人民币 |

**关键经历**：
- 高中毕业后参军（工程兵）
- 1973年以数理化全县第二名考入中科大
- 1976年毕业后分配至中科院武汉物理研究所，从事超导磁体研究
- 1980年赴法国巴黎第十一大学访问进修
- ~1986年回国后被调至深圳安科（中科院与美国Analogic合资，被誉为"医疗器械黄埔军校"），任办公室主任
- **1991年**：40岁，不满安科体制束缚，辞职创办迈瑞

**关键决策与战略**：
- 确立"研发投入≥营收10%"的铁律，延续至今
- 2006年：主导纽交所上市（中国首家海外上市医疗器械企业）
- 2016年：主导33亿美元私有化退市，回归A股
- 2018年：深交所创业板上市，首日市值854亿元
- 2025年：筹划赴港二次上市（千亿级IPO）
- 战略目标：进入全球医疗器械Top 10（目前约Top 30）

**公益捐赠**：
- 2020年：向中科大捐赠1.068亿元，成立"李西廷基金"
- 2022年：向母校砀山中学捐赠10亿元建设新校区

**争议**：
- 2018年IPO前入籍新加坡，"老板移民"引发舆论不满
- 控股架构为"BVI离岸公司→新加坡免税"，被质疑避税
- 多年大额减持，累计套现数百亿

## 三、管理层稳定性与变动

### 3.1 "三剑客"时代落幕 —— 创始人交班

这是迈瑞近年最重大的管理层变动：

- **2025年10月**：副董事长成明和卸任副董事长，降为普通非执行董事，仅提供战略建议。媒体标题："迈瑞三剑客，只剩下74岁的李西廷坚守了"
- **徐航**早在2012年辞去联席CEO，已长期不参与日常管理
- **李西廷**（74岁）被问到退休计划时**未正面回答**，媒体评论"眼下的迈瑞确实离不开他"

### 3.2 与前高管的诉讼纠纷（2026年）

迈瑞赴港IPO期间与**4位前高管**因有限合伙（员工持股平台）纠纷对簿公堂：

| 前高管 | 曾任职务 | 开庭时间 |
|--------|---------|---------|
| 徐超 | 全球营销系统副总经理 | 2026年1月16日 |
| 潘蒂 | 前高管 | 2026年3月5日 |
| 陈朝 | 国际法律总监 | — |
| 栾晨光 | 2015-2017任职 | — |

案由均为"有限合伙纠纷"，涉及员工持股平台权益争议。2026年5月10日招股书失效。

### 3.3 员工持股计划（已部分失败）

| 项目 | 详情 |
|------|------|
| 推出时间 | 2022年1月 |
| 参与人数 | ~2,500-2,700名核心员工 |
| 规模 | 回购约305万股，过户价50元/股 |
| 解锁安排 | 三期：24/36/48个月 |
| 业绩目标 | 以2021年归母净利80亿为基准：2024年≥+73% |
| 实际结果 | **第三批❌未达标**（2024年仅增45.8%，远低于73%目标） |
| 后果 | 第三批全部不得解锁，冲回已确认股份支付费用约1.04亿元 |

### 3.4 管理层集体增持（2025年底）—— 信心信号

- **李西廷**：2025年11-12月累计增持100.54万股，金额达**2亿元**，承诺锁定6个月不减持
- 多位核心高管集体跟进增持
- 公司完成**10亿元股份回购**，324.99万股将依法注销

### 芒格视角点评

> *"以合理的价格买入一家优秀的公司，远好过以优秀的价格买入一家合理的公司。"*

迈瑞的管理层无疑是优秀的——三位创始人从安科的体制束缚中出走，在几乎空白的中国医疗器械市场中，用30年打造了全球Top 30的医疗器械帝国。他们的战略眼光（逆境收购Datascope、私有化回A、高强度研发投入）令人钦佩。

但另一方面，段永平说过"企业文化就是'利润之上'的追求"。迈瑞创始人**巨额减持**与**高薪**之间的张力，**承诺期后立即减持**的诚信问题，以及**74岁创始人未明确交班**的传承风险——这些都是投资者需要认真权衡的因素。

巴菲特说过："我们最喜欢的持有期是永远。"但迈瑞创始人显然不这么想——他们用行动告诉我们，"永远"可能只是投资者的幻想。
